Introduction to HTML

Short answer type questions -

Qu. 1 - What is HTML?

Ans. - HTML stands for Hypertext Markup Language. It is a markup language that
describes the structure of the web page. It allows us to create web pages that contain text
and graphics.

Qu. 2 - Name the types of tags in HTML?

Ans. - HTML provides various tags such as

  1. <HTML>
  2. <HEAD>
  3. <TITLE>
  4. <BODY>
  5. <FONT> etc.

Qu. 3 - Define

            (i) Hypertext (ii) Markup language

Ans. - Hypertext-

                            Hypertext is a piece of ordinary text that has special feature of linking
                            to other documents or web pages.

          Markup language-

                            Markup language that uses special symbols called Tags to mark up
                            text document that instructs the browser how to display the text.

Long Answer type questions -

Qu. 1 - Explain the purpose of <HTML> and <BODY> tags.

Ans. <HTML> tag -

                                  The <HTML> tag tells the web browser that text contained
                                  between <HTML> and </HTML> is a web page and can be viewed
                                  using a web browser. Every web page coding must start with the
                                  <HTML> tag and end with the </HTML> tag.
